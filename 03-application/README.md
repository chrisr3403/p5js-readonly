## Conditionals

You can use conditionals in P5js just like you normally would in JS:

```js
if(condition) {
    // code to run
} else if(condition) {
    // code to run
} else {
    // code to run
}
```

As always, you _must_ have an `if`, as many `else if`s as you want, and at most _one_ `else`.
